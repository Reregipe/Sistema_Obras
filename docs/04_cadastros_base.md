# Cadastros Base

- Equipes
  - Campos: codigo, linha (viva/morta), encarregado, telefone, componentes, ativa?
  - Usado em acionamento, execucao, medicao, faturamento.

- Materiais
  - Campos: codigo, descricao, unidade, status (ativo/inativo).
  - Usado em listas de aplicacao/medicao.

- Mao de obra (codigos MO)
  - Campos: codigo, descricao, ups, status.
  - Usado para apontar servicos executados.

- Viaturas
  - Campos: placa, modelo, apelido, status.
  - Usado em acionamento/execucao.

Observacao: cadastros servem a todos os fluxos (acionamento, obras, faturamento).
